Setup a remote repository in Local , Add a file and commit or save the changes and push to master branch


$ mkdir Project_1

$ cd Project_1

$ echo "This file is directly pushed to git using terminal" >  msgfile.txt

$ ls

$ git init

$ git status

$ git add msgfile.txt

$ git commit -m "Git file added"

$ git remote add origin https://github.com/harshitgarg123/project.git

$ git push -u origin master
